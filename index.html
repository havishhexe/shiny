How to use

01

Add the script

Drop this before your closing </body> tag.

<script src="https://cdn.jsdelivr.net/npm/shine-links@latest/shine.js"></script>
02

Apply the effect

Add one line of JS to apply shine to all links at once.

document.querySelectorAll("a")
  .forEach(el => el.classList.add("shine"));
03

Or just add the class

Add shine directly to any element in HTML.

<a class="shine" href="#">My link</a>

Ways to use it

All links on the page

Apply shine to every single link automatically with one line.

document.querySelectorAll("a")
  .forEach(el => el.classList.add("shine"));

Only nav links

Target a specific group using any CSS selector.

document.querySelectorAll("nav a")
  .forEach(el => el.classList.add("shine"));

A single element

Add the class directly in HTML — no JS needed at all.

<a class="shine" href="/about">About</a>

Any element, not just links

Works on headings, buttons, spans — anything you want to shine.

<h1 class="shine">Hello world</h1>

Note on shine color

Match your shine color to your background.

On a dark background — increase the white opacity in the gradient, e.g. rgba(255,255,255,0.9)

On a light background — keep opacity lower so it's subtle, e.g. rgba(255,255,255,0.5)

For colored text like gold — use a matching color in the gradient to keep it natural.

Customization

If you're using the CDN you can't edit shine.js directly — override these in your own CSS instead.

.shine::after {
  animation-duration: 2.2s;   /* speed — lower = faster */
  animation-delay: 0s;        /* delay before first sweep */
  width: 50%;                 /* streak width — lower = sharper */
  transform: skewX(-15deg);   /* tilt — 0 = straight */
  background: linear-gradient(
    105deg,
    transparent 30%,
    rgba(255,255,255,0.55) 50%, /* change color + opacity here */
    transparent 70%
  );
}

Speed

Lower = faster sweep. Default is 2.2s.

.shine::after { animation-duration: 1s; }

Color

Match the streak color to your text. Keep opacity around 0.5.

.shine::after {
  background: linear-gradient(105deg,
    transparent 30%,
    rgba(255,200,0,0.5) 50%, /* gold */
    transparent 70%
  );
}

Streak width

Lower = thinner, sharper streak. Default is 50%.

.shine::after { width: 25%; }

Tilt

Controls the angle of the streak. 0 = straight, -15 = angled.

.shine::after { transform: skewX(-5deg); }

Delay

Useful for staggering shine on multiple elements.

.shine::after { animation-delay: 0.5s; }

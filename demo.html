How to use

01

Add the script

Drop this before your closing </body> tag. 

<script src="https://cdn.jsdelivr.net/npm/shine-links/shine.js"></script>
02

Apply the effect

Add one line of JS to shine all links at once. 

document.querySelectorAll("a")
  .forEach(el => el.classList.add("shine"));
03

Or target a specific element

Just add the class directly in your HTML.

<a class="shine" href="#">
  <span class="s">My link</span>
</a>

Ways to use it

All links on the page

Apply shine to every single link automatically. 

document.querySelectorAll("a")
  .forEach(el => el.classList.add("shine"));

Only nav links

Target a specific group using any CSS selector. 

document.querySelectorAll("nav a")
  .forEach(el => el.classList.add("shine"));

A single element

Add the class directly in HTML — no JS needed. 

<a class="shine" href="/about">
  <span class="s">About</span>
</a>

Any element, not just links

Works on headings, buttons, spans — anything. 

<h1 class="shine">
  <span class="s">Hello world</span>
</h1>

Note on shine color

Match your shine color to your background.

On a dark background — use a white streak with mix-blend-mode: overlay

On a light background — use a white streak with mix-blend-mode: screen so it's invisible on the bg but lights up the text.

For colored text like gold — use a matching color streak with mix-blend-mode: overlay to keep it natural.

shiny

cartoon shine for your links — 20 lines of js

github.com/havishhexe/shiny
01

Add the script

Drop this before your closing </body> tag. 

<script src="https://cdn.jsdelivr.net/npm/shine-links/shine.js"></script>
02

Apply the effect

Add one line of JS to apply shine to all your links.

document.querySelectorAll("a")
  .forEach(el => el.classList.add("shine"));
03

Style your links

Remove the default blue so it actually looks good.

a {
  color: #0a0a0a;
  text-decoration: none;
}
